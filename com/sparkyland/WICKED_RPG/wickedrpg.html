WICKED RPG is a role playing game being developed by Spartique. It is currently playable, but unfinished.

WICKED RPG is a cute little overhead role playing game. Both versions of the game are esentially the same game. The difference is that Java 1.2 supports MIDI, a compact music format that is ideal for the web. WICKED RPG 1.2 has awesome music and sound effects! WICKED RPG1.1 has very little audio.

PLAY WICKED RPG 1.1 - Java 1.1 is required. This is available on most browsers.
WICKED RPG 1.2 - Java 1.2 is required. This is not available on most browsers. If you click this link, and you do not have Java 1.2, you will be asked to download a 9 Megabyte plugin from Sun Micorsystems, the makers of Java. It's very easy to install, and you only have to do it once. If you do, you will be able to run more Java 1.2 programs on the web! (Currently only works with Microsoft Internet Explorer). No reboot is necessary.
